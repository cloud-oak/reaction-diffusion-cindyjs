Runge-Kutta

  • $\dot{y} = f(y), \quad y_0$ gegeben
  • $k_1 = f(y_n)$
  • $k_2 = f(y_n + \frac{h}{2} k_1)$
  • $k_3 = f(y_n + \frac{h}{2} k_2)$
  • $k_4 = f(y_n + h k_3)$
  • $y_{n+1} = y_n + \frac{h}{6}(k_1 + 2 k_2 + 2 k_3 + k_4)$
